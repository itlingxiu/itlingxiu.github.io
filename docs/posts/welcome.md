# 欢迎来到光影博客

欢迎来到光影博客！这是一个基于 VitePress 构建的现代化博客网站。

## 博客特色

### 🚀 高性能
- 基于 VitePress 构建，享受极速的页面加载体验
- 静态站点生成，SEO 友好
- 支持渐进式 Web 应用 (PWA)

### 🎨 现代化设计
- 简洁优雅的界面设计
- 支持深色/浅色主题切换
- 响应式布局，完美适配各种设备

### 📝 丰富功能
- 支持 Markdown 语法
- 代码高亮显示
- 全文搜索功能
- 分类和标签管理

## 内容规划

这个博客将主要分享以下内容：

1. **技术文章** - 前端开发、后端技术、工具使用等
2. **设计分享** - UI/UX 设计、视觉设计相关
3. **学习笔记** - 技术学习心得和总结
4. **生活感悟** - 个人思考和生活记录
